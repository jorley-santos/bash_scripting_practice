# Backup-Folder


Objective: This repository was created to store scripts I used to practice some basic Bash Shell operations. The main script (/bin/backup.sh) is used to create an executable backup command in Bash. This command creates a copy of any folder inserted by the user, according the following structure:

$ backup \<folder>

A new folder will be created to store the copy generated by the script. It is not a production level code by any means.

Requirements: Bash Shell
